## Hot Keys:
- F - Fill shape
- Mouse3 - change shape:
    - Circle
    - Rectangle
    - Triangle(rectangle don't ask why)
- S - Save shape picture in .dat
- L - Load shapes.dat
- Y - Choose a color in terminal in RGB range (0 -> 255) 

 <b> For compile use </b> ```make```
